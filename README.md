## About

I'm a 4th-year BSIT student at the **Technological University of the Philippines – Taguig** and a full-stack developer who builds products end-to-end — web, mobile, and hardware. I work across the **MERN stack** (MongoDB, Express, React, Node) and **Laravel / PHP**, with **HTML, CSS**, and **SQL / SQLite / MongoDB** on the data side, and I ship mobile apps with **React Native and Expo Go**. Alongside that, I build on **Stellar** and **Ethereum**, work with **ESP32 / Arduino** for IoT, and create short-form video content that I edit and render myself.

- **Full-stack web development** — MERN stack, Laravel / PHP, HTML, CSS
- **Databases** — SQL, SQLite, MongoDB, PostgreSQL, Supabase, Firebase
- **Mobile app development** — React Native with Expo Go
- **Blockchain and Web3** — Stellar payments and smart-contract applications
- **IoT and embedded systems** — ESP32 and Arduino
- **AI integration** — chatbots and features built on top of LLM APIs
- **Content creation** — short-form video, editing, and rendering
